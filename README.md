# Agency Pipeline

Collaborative repo for the agency team — audits, demo sites, lead management, and outreach automation.

## Structure

- `/audits` — Site audit reports (speed, SEO, mobile, accessibility)
- `/templates` — Demo site templates by niche (dental, legal, salon, etc.)
- `/leads` — Lead lists, CSVs, and dedup tracker
- `/outreach` — Email copy, sequences, and campaign configs
- `/scripts` — Scrapers, automation tools, utilities
